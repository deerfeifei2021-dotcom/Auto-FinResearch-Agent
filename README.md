# 🚀 Auto-FinResearch-Agent

基于多 Agent 协同与长链推理的企业级财报深度分析与风险预警系统 (Proof of Concept)

## 💡 项目背景与核心痛点
金融与法务团队在进行企业尽调时，面临海量且超长篇幅的财报、公告及新闻。人工审查不仅效率极低，且极易遗漏隐藏在长文本中的合规隐患、对赌协议及数据异常风险。

## 🧠 核心逻辑流 (多 Agent 协作)
本项目采用多 Agent 流水线架构，旨在处理十万字级别的超长上下文：
1. **Data Fetcher Agent**: 自动化拉取多源长文本数据。
2. **Financial Analyst Agent**: 结合 RAG 技术，对长文本进行数据提取与环比逻辑计算。
3. **Risk Auditor Agent**: 专注于长链逻辑推演，跨段落寻找诉讼隐患与合规风险。
4. **CIO Agent**: 汇总多源信息，输出最终决策报告。

## 📈 Token 消耗预期
当前项目已完成核心链路的终端测试。由于目标处理对象均为超长文本（财报/法律合同），且涉及多个 Agent 节点的回合制推理（Debate），预期在实际业务落地后，**单次分析将消耗大量 Token，日均 Token 消耗量极高**。目前正在申请更高额度的 API Plan 以支撑下一步的并发压测与业务接入。
